javascript 코드라이언 강의 수강

javaScript로 만드는 [로또 번호 추첨기]

  1. 여러 개의 값 담기
    • 배열 활용
    • var lotto = [];
  2. 번호 무작위로 추첨하기
    • Math.random() * 45 + 1
    • * 45 : 0 이상 45미만
    • + 1 : 1 이상 46미만
  3. 번호 중복되지 않게 하기
    • if ( lotto.indexOf(num) == -1 ) { lotto.push(num) }
    • 특정 문자열, 숫자가 배열에 있는지 확인해주는 indexOf 활용
    • -1 : lotto 배열에 num 없음
    • 중복되지 않은 num만 lotto 배열에 push됨
  4. 중복되지 않은 숫자가 총 6개 될때까지 반복하기
    • while : 특정 조건까지 계속 반복
    • while (조건) {반복하려는 코드}
    • while (lotto.length < 6)
    • 왜 7이 아닌, 6이냐 ? 배열의 인덱스는 0부터 시작하기 때문 !
  5. 로또 번호를 오름차순으로 정렬하기
    • .sort() : 배열 값 정렬
    • 사전순 정렬 : [1,11,2,22,3,33]
    • 오름차순 정렬 : .sort((a,b) => a-b)
    • 내림차순 정렬 : .sort((a,b) => b-a)

기념일 계산기 만들기

  1. 객체
    • content.length, console.log -> 모두 객체 !
  2. Date 객체
    • new Date()
    • now.getFullYear() : 현재 년도
    • now.getMonth() : 현재 월 - 0 (1월)
    • now.getDate() : 현재 일
    • now.getTime() : 현재 시간, 1970/1/1/0:0 기준으로 흐른 시간을 표시
  3. 며칠째 사귀고 있는지 기능 구현
    • 만난 밀리초 = 오늘.getTime - 사귄날.getTime()
    • 밀리초 / (1000 * 60 * 60 * 24) + 1
    • 정수로 Math.floor()
  4. 기념일까지 며칠 남았는지 기능
    • 기념일 밀리초 = 기념일.getTime() - 오늘.getTime()
  5. 1000일까지 얼마나?
    • var ms = 1000일의 밀리초 = 사귄날.getTime() + 999일.gettime()
    • thousand = new Date(ms)
    • thousand.getFullYear()
    • var thousand = new Date(start.getTime() + 999 * (1000 * 60 * 60 * 24));
