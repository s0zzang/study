모달창 만들기

  1. 모달 레이아웃
    • .modal
    • .modal-inside : 가운데 정렬
    • .window : 여기가 변경됨 (확장 가능성 고려)
  2. 모달 자체 스크롤바 생기기
    • 모달 min-height: 100vh; 인 경우 스크롤 바가 모달 자체에 생기지 않음 .. !
    • 이거 해보기 !!
  3. 모달 on/off
    • off - opacity: 0, pointer-events: none
    • visibility가 아닌 pointer-events를 사용한 이유 : 보여도 없는 것처럼 취급할 수 있음
    • tab키로 도달은 가능함
    • visibility: hidden; : tab키로 접근 불가
    • inert : 해당 요소 비활성화 상태로 만들어 - 모달창 제외 컨테이너한테 주면 좋을 듯 ?
  4. 모달 크기 커져서 스크롤바 생길때 안덜컹거리기
    • modal > .modal-resize 하나 더 만들어서 100vw 줌
    • 별도 적용 예제 참고하기
  5. body도 스크롤, 모달도 스크롤이 있을 때 - 이중 스크롤
    • 스크롤 체이닝
    • overflow: auto 준 애한테 overscroll-behavior: none
    • 가로 스크롤한텐 안주는게 좋음
    • 사용자는 오히려 오류처럼 느낄 수 있음
  6. body는 스크롤, 모달은 스크롤 없을 때 (위에 안주고 이것만줘도 됨)
    • body에 overflow: hidden;
    • body:has(.is_active)

  7. 스크롤바 미리 주고 안보이게 처리하기
    • .modal::-webkit-scrollbar { w20 h20 }
    • .modal::-webkit-scrollbar:enabled { wunset }
