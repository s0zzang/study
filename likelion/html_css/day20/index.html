form 요소의 기본 구조 / 기능 살펴보기

  1. form 전송
    • button type="submit" 클릭시 하여 제출
    • forma action="주소" 어디로 보낼지 작성
    • 백에서 받을 준비된 주소를 작성함
  2. filedset
    • 그룹핑
    • 테두리 생김
  3. legend
    • filedset 그룹의 제목
  4. label
    • 클릭 가능 요소
    • 클릭 했을 때, 해당 인풋에 포커스 잡히게 하기 : label for = input id
  5. input 활용
    • input 사용시, ... 처리해도 ... 이전 텍스트까지 복사 가능
    • placeholder : 입력 값에 대한 힌트 제공
    • readonly : 읽기 전용
    • required : 필수값 지정 - 브라우저마다 스타일 다름

form요소 스타일링 진행하기

  1. 브라우저 간 차이
    • 특히 form 관련 요소가 다른 점이 상당히 많음
    • 기본 스타일 초기화 작업 우선 !
    • all: unset, display: block
    • input, button : inline-size: 100% (button은 생략 가능)
  2. input normal
    • min-inline-size : min-width와 동일 = width 100%로 사용 가능
    • height 지정 => line-height 없어도 세로 중앙정렬됨
    • padding: 0 12px
    • input 크기 조정하고 싶을 때 > input을 감싼 div를 컨트롤 (보더, width
    • .form-input input:focus { border-color: red }
    • .form-input:focus-within - 내부 모든 input이 포커스 되었을 때 부모 선택
