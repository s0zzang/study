display grid 입문

  • 1. flex와 비슷한 컨셉
  • 부모에게 display:grid - grid container
  • 자식은 이제 grid item
  • gap 사용 가능
  • 부모한테 height 500px -> grid item들이 500을 나눠가짐
  • 2. flex와 다른점
  • grid는 선언시 가로 정렬되지 않음
  • 한방향인 플렉스와 다르게 양방향 설정 가능
  • 단점) flex-direction: row-reverse 이런거 없음
  • 3. 가로축 나누기
  • grid-container 한테 설정
  • grid-template-columns:100px : 100px 너비 grid item 1개
  • grid-template-columns:100px 100px : 100px 너비 grid item 2개
  • fr : fraction(분수) 단위 활용
  • grid-template-columns:1fr : 1 비율
  • grid-template-columns:1fr 3fr auto : 1:3:콘텐츠 비율
  • grid-template-columns:1fr 1fr 1fr : 1:1:1 비율
  • 4. grid item 추가
  • item이 추가되면, 자동으로 틀 안에 추가됨
  • container의 height 안에서 영역을 나눠가짐
  • 5. grid 사용해보기
  • grid-template-columns:repeat(3, 1fr) : 1fr을 3번 반복하겠삼
  • grid-template-columns:repeat(3, minmax(200px, 1fr)) : 최소 200px, chleo 1fr
  • grid-template-columns:repeat(auto-fit, minmax(200px, 1fr)) : auto-fit : 200px이 동작하는 기준이 됨
  • 6. auto-fit, auto-fill
  • grid-template-columns:repeat(auto-fit, minmax(20px, 1fr))
  • auto-fit : 20px씩 채우고 남은 공간을 1fr에 균등분배하여 여백 없음
  • grid-template-columns:repeat(auto-fill, minmax(20px, 1fr))
  • auto-fill : 20px씩 채우고 남은 공간을 가상의 셀로 채워 여백으로 놔둠
  • 줄바꿈 발생하면 둘다 동일해보임
