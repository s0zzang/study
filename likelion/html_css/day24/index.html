  2. 구조
    • thead, tbody, tfoot
  3. caption
    • 테이블 설명
    • 웹표준 : table > caption 첫번째 요소로 들어가야 함
  4. border-collapse
    • 보더 병합 관련 속성
    • 기본값 : separate
    • border-spacing 간격으로 설정 가능
    • 보통 많이 사용하는 값 : collapse
    • border-spacing 의미 없어짐
    • 보더 역시 병합되어 노출됨 - border-width가 높은 쪽으로 병합됨
    • table border와 td,th의 border도 병합 현상이 발생함
  5. cell 너비
    • 셀 내부 콘텐츠에 따라 알아서 너비를 분배함
  6. table-layout
    • 초기값 auto - 텍스트 양만큼 알아서 결정
    • fixed - 텍스트 양이 아닌, 균등 분배 - cell 하나의 너비 지정하면 나머지는 그 안에서 또 균등 분할
  7. colgroup > col
    • <col> 셀프 클로징 태그
    • 콜을 그룹으로 묶어서 한번에 선택 가능 (1번째 셀 모두)
    • <col class="type_small"> small, medium, large 등으로 스타일 지정

테이블요소를 스타일링 하기 위한 변수 확인하기

  1. table
    • border-collapse: separate(초기) -> collapse
    • border-spacing: unset
    • width: 100%
    • table-layout: auto -> fixed
    • height - min-height처럼 동작함 -> 이것보단 각 셀에 padding 방법 더 사용함
  2. th, td
    • padding: 1px > 원하는 값
  3. colgroup > col
    • col class="type_spotlight" span="2" : 이런식으로 2개의 col에 한번에 클래스 적용 가능
