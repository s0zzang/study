html, css 3일차 강의

상대경로와 절대경로
내 위치 : ./ (생략가능)
상위폴더 : ../
절대경로 : / (document route) - 최상위 폴더에서 경로 탐색
권장되는 방식 : 적재적소에 맞춰 사용
reset 관련
UA 스타일을 초기화하기 위해 initial이 더 권장됨
ex) body { margin: 0 } => body { margin: initial }
initial : 개발자의 의도가 분명하여 reset시 권장되나, 익스플로러 지원이 안되기 때문에 주로 0을 사용함
unset : 사용시 프로퍼티의 상속 여부에 따라 다른 값을 지정해줌
- 프로퍼티가 상속 O : inherit 지정
- 프로퍼티가 상속 X : initial 지정
시멘틱 태그
main: h1과 같이 문서에 한번만 작성 가능
header, footer 등은 여러번 사용 가능
마진 병합 현상
이미지 하단 여백 생기는 현상
Class Selector
(*).class_name { ... } : 클래스 선택자 앞 * 가 생략되어 있음
combinator : 결합자, 셀렉터 사이에서 새로운 가치를 만들어 내는 역할 (' ', >, +)
하위 셀렉터: 상위요소 하위요소 (띄어쓰기 : descendant combinator)
자식 셀렉터: 부모 > 자식 (> : child combinator)
인접 형제 셀렉터: 형제 + 형제 (> : sibling combinator)
일반 형제 셀렉터: 형제 ~ 형제 (> : sibling combinator)
