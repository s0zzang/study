캐스케이딩 Cascading
위에서 아래로 떨어지는, 폭포
스타일링(최종 결과물)에 적용되는 일련의 과정, 모든 알고리즘 = css의 근간 (Cascading Style Sheet)
Selector Specificity 정도, 상속 여부, ...
width: auto, width: 100%
width: auto : 컨테이닝 블록 기준으로 가득 채워지짐 - 보더, 마진, 패딩 등을 브라우저가 계산하여 가득 채워짐
width: 100% : 컨테이닝 블록 기준으로 100%가 꽉 채워지짐 - 보더, 마진, 패딩 등이 고려되지 않고 옆으로 밀림, 튀어나감
inline-block 요소간 줄바꿈(띄어쓰기) 관리
극단적-비추) 부모: font-size:0, 자식: font-size:16px
추천) 플렉스 박스
line-height
font-size * line-height = 컨텐츠 높이
initial value = normal : 폰트 종류에 종속됨 (폰트 만드는 사람들이 만드는 것)
텍스트 기준 상하 여백(Half Leading area)이 생김
Half Leading area - lead 납 = 줄 간격, 활자에서 상하 여백을 주기 위해 끼워넣던 lead(납) 용어가 그대로 이어짐
