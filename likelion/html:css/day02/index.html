Block-level elements vs inline element
inline : 박스가 아닌 텍스트 개념, width가 아닌 length 개념이 적용됨
inline-block : inline과 block의 특징을 모두 가지고 있음
inline은 안되지만 inline-block은 되는 것 : width, height, padding/margin 상하
