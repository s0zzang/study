CSS의 동작 방식 이해
css의 목적 : 웹문서 꾸밈
타입 셀렉터 Type Selector
h1 { ... } 에서 h1 선택자를 뜻함
* { ... } : 전체 선택자 Universal Selector
초기값 initial value과 브라우저 기본 스타일
h1의 backgorund-color: transparent와 같은 초기값 - 스펙 (브라우저 무관), mdn 참고
요소별로 적용된 user agent stylesheet : 브라우저 기본 스타일
스타일 상속 여부 : 개발자 도구 - 상속 활성화, mdn 참고
CSS Box-model 입문
margin: auto 관련 - 할당가능한 영역을
margin left, right : auto 적용 가능, 상위 컨텐츠가 가지고 있는 여백만큼 적용 가능
margin top, bottom : auto 적용 불가능, 상위 컨텐츠가 가지고 있는 상하 여백이 없어서 불가능
containing block: 직계 부모가 제공해주는 컨텐츠 영역
용어 정리
html 속성 : 어트리뷰트 - 정적인 값
css 속성 : 프로퍼티 - 동적인 값, 자스에서 사용됨
  • margin의 auto 값이 어떻게 적용되는지? : 직계부모가 제공해주는 영역에서 할당가능한 영역만큼 정렬
  • margin과 padding의 차이 : margin은 요소 외부의 영역, padding은 요소 내부의 영역
  • css에서 상속이란? : 상위 요소에서 지정한 스타일이 하위 요소에도 적용되는 것
  • css 프로퍼티의 기본값 : 태그 자체에 부여된 initial value와 브라우저마다 각각 적용된 user agent stylesheet 값
