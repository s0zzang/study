  • 속성 값 가져오기
  • A.attr('b') : a요소의 b라는 속성의 값 가져오기
  • var c = A.attr('b') : a요소의 b라는 속성값을 가져와서 c 변수에 저장
  • 예시) $btn.find('img').attr('src')

  • A.attr('b','c') : a요소의 b라는 속성을 c로 변경하기
  • var c = A.attr('b') : a요소의 b라는 속성값을 가져와서 c 변수에 저장
  • 예시) $btn.find('img').attr('src')
